# Freepik Downloaded Assets Organizer
A node.js script to organize downloded files from Freepik

## Directory structure

Input directory structore
```
- downloads
  - person-illustration
    - 1.eps
    - 1.jpg
  - city-sketch
    - 401.ai
    - 401.jpg
    - 404.eps
    - 404.jpg
```

Input directory structore
```
- organized
  - person-illustration
    - person-illustration-001.eps
    - person-illustration-001.jpg
  - city-sketch
    - city-sketch-001.ai
    - city-sketch-001.jpg
    - city-sketch-001.eps
```
